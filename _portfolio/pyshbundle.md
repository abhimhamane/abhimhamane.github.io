# Explore. Collaborate. Discover.

PySHBundle is the python implementation of the popular SHBundle toolbox originally written using MATLAB.

## Details

There is an existing MATLAB code bundle named SHbundle, which was developed by Sneew et al. in 2021 and is distributed under the GNU license. This code bundle is open for free use and modification by anyone, provided that proper credit is given to the original developers. However, it's worth noting that due to MATLAB's proprietary nature, there might be limitations in terms of accessibility. Below, we provide a brief overview of the impact of the SHBundle package.

Conversely, there's a vibrant community of Python programmers, champions of the open-source programming world. In our endeavor, we've taken the MATLAB codes from the SHbundle and translated them into the Python programming language. It's worth noting that Python boasts an ever-expanding array of geospatial processing capabilities.
