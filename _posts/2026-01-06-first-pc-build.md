So finall all the parts arrived!

# OS installation

This was really unexpected. I wanted to do a cinebench and other testing of my new components, so went with installing Windows 11. I skipped the network requirement with `oobe\bypassnro` command and everything went smooth. The catch was I had no ethernet, WiFi and bluetooth didn't work. I do have a WiFi motherboard, but windows does not come with those drivers and one needs to install it seperately from the OEM website.

Later, I tried installing Fedora and Mint. Suprisingly, they had no issues with WiFi drivers. Though, I had some issue with bluetooth earbuds not connecting, or if connected they had a weird buzzing ring as if it might explode. Nthing happened and this issue vanished after restarting and updating the system.

Out of box experience, I was suprised with linux working so good. I had faced bluetooth issue many times on my old laptop, but it seems there has been some improvement.
